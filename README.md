# Signal v2

Your AI briefing, zero noise. Personalized updates, audio overview, and tool recommendations.

- **signal-app/** — Next.js app (onboarding, RSS-backed briefing, Ask Signal chat, AI Tool Recommender, audio overview). Deploy with `npx vercel` from inside `signal-app/`.
- **prd/** — Product requirements (Signal + AI Tool Recommender).
- **context.bin** — Original hackathon context archive.

## Quick start

```bash
cd signal-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and complete onboarding to see your personalized briefing and audio overview.
